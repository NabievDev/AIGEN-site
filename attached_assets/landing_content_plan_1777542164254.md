## 3. Анимации и интерактив

### Загрузка страницы (page load)
- Логотип появляется с `opacity: 0 → 1` + `translateY(20px → 0)`, задержка 100ms
- Главный заголовок: посимвольный reveal (JS typewriter effect) — каждое слово появляется с stagger 80ms
- Подзаголовок и CTA-кнопка: `fadeIn + translateY` с задержкой 600ms и 800ms
- Фоновые blob-элементы: медленное `float` (CSS keyframes, период 8–12s)

### Scroll-анимации (Intersection Observer)
- Все секции: `opacity: 0 → 1` + `translateY(40px → 0)` при входе во viewport
- Карточки преимуществ: stagger-появление (каждая следующая с задержкой +100ms)
- Карточки расписания: появление слева-направо поочерёдно
- Счётчики (цифры статистики): анимированный countUp при попадании в зону видимости
- Таймлайн месяцев: линия «рисуется» сверху вниз при скролле (stroke-dashoffset)

### Микровзаимодействия
- CTA-кнопки: scale(1.04) + glow-тень при hover, scale(0.97) при click
- Карточки: translateY(-6px) + усиление тени при hover
- Навигационные ссылки: underline-анимация слева направо при hover
- Аккордеон FAQ: плавное раскрытие с `max-height` transition
- Мобильное меню: slide-in справа с backdrop blur

### Фоновые эффекты
- Hero: canvas с медленно движущимися частицами (50–80 точек, opacity 0.3)
- Gradient blob 1: `#7C3AED` с blur 120px, анимация float
- Gradient blob 2: `#06B6D4` с blur 100px, анимация float с задержкой
- Dot grid overlay на светлых секциях: SVG pattern, opacity 0.04

## 7. Технические требования для Replit

- Один HTML-файл `index.html` — чистый HTML5
- Стили: встроенный `<style>` или отдельный `style.css`
- Скрипты: встроенный `<script>` или `script.js`
- Шрифты: Google Fonts через `<link>` (`Syne`, `Onest`, `Space Mono`)
- Иконки: Lucide Icons через CDN (`unpkg.com/lucide@latest`)
- Анимации: чистый CSS + ванильный JS (без тяжёлых фреймворков)
- Canvas-частицы: нативный Canvas API (~50 строк JS)
- Scroll-анимации: Intersection Observer API (нативный, без библиотек)
- Яндекс Форма: `<a href="[ССЫЛКА]" target="_blank">` на кнопку или `<iframe>`
- Адаптивность: mobile-first, breakpoints 768px и 1024px
- Производительность: изображения в WebP, шрифты с `display=swap`, `loading="lazy"` на картинках
